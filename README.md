# Recordings Tracker

Track various recordings of events my family performed in. For example, band concerts, etc.

# Requirements

- Python 3+
- pip
- Django 5.1+
- Django REST Framework

# Setup

```sh
$ python -m venv .
$ . .venv/bin/activate
(.venv) $ python -m pip install -r requirements.txt 
```
